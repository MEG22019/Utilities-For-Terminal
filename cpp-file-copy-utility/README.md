# File Copy Utility

A simple C++ command-line tool that copies the contents of one file into another.

The program prompts the user for a source file and a destination file, then copies the contents line-by-line.

## Features

- Reads from a source file
- Creates a destination file
- Copies contents line-by-line
- Basic error handling if files cannot be opened

## Compilation

Compile using g++:

```bash
g++ autosave.cpp -o copy
